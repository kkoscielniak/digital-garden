This [[about/digital-garden|digital garden]] is built using [VitePress](https://vitepress.dev).

## Plugins

My gardening setup is basically pure VitePress with several plugins like:

- `markdown-it-checkbox`
- `markdown-it-wikilinks`
- `vitepress-sidebar`
