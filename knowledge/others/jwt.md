---
title: JSON Web Token
---

```
AAAA.BBBB.CCCC
```

- `AAAA` - _header_. Type of token (JWT), and the signing algorithm used
- `BBBB` - _payload_. Claims about the user and additional data
- `CCCC` - _signature_. Used to verify if the token wasn't malformed.
